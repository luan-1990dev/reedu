# Reedu - Reeducação Alimentar Inteligente 🍎🚀

O **Reedu** é um aplicativo completo de acompanhamento nutricional desenvolvido em **Flutter** e **Firebase**. Ele foi projetado para transformar a rotina alimentar em uma experiência automatizada e analítica.

## 🌟 Funcionalidades Principais

- **🏠 Home Inteligente:** Sugestões de refeição em tempo real baseadas no horário do sistema Android.
- **📊 Tendência de Peso:** Gráfico interativo com histórico de datas capturadas automaticamente.
- **📄 Importação via PDF:** Leitura automática de Planos Alimentares, Suplementação e Cardápios direto de arquivos PDF.
- **📝 Gestão de Cardápio:** Seleção rápida de opções via Dropdown e registro de consumo diário.
- **📈 Inteligência de Dados:** Ranking das refeições "Mais Consumidas" com limpeza automática a cada 30 dias.
- **⏰ Lembretes Automáticos:** Notificações de refeição (10 min antes) e avisos de hidratação a cada 3 horas.
- **🛡️ Segurança:** Autenticação via Google e Biometria (Impressão Digital).
- **📸 Perfil Personalizado:** Avatar com upload de foto da galeria e modo imersivo total (Edge-to-Edge).

## 🛠️ Tecnologias Utilizadas

- **Framework:** Flutter
- **Linguagem:** Dart
- **Backend:** Firebase (Firestore / Auth)
- **Gráficos:** fl_chart
- **PDF Engine:** syncfusion_flutter_pdf
- **Notificações:** flutter_local_notifications

## 🚀 Como Executar o Projeto

1. Certifique-se de ter o Flutter instalado em sua máquina.
2. Clone este repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/reedu.git
   ```
3. Instale as dependências:
   ```bash
   flutter pub get
   ```
4. Execute o app:
   ```bash
   flutter run
   ```
